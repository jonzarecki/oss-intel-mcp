MCP Tools

Comprehensive health report with an overall verdict: Safe to adopt, Use with caution, or Risky. The core intelligence tool.

Contributor-focused analysis. Surfaces PR acceptance rate, review speed, maintainer responsiveness, and community health signals.

Two Questions This Server Answers

🛡️ "Should I use this?"

Is this dependency safe, maintained, and well-backed? Get a 0-100 verdict score based on activity trends, bus factor, PR health, issue responsiveness, release cadence, and corporate backing — far beyond star counts and last commit dates.

🤝 "Should I contribute?"

Will your PRs get reviewed? Is the community welcoming? See PR acceptance rates, median review times, maintainer responsiveness, good-first-issue availability, and contributor retention — everything you need before investing your time.
